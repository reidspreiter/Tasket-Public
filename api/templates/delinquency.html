Delinquency

The delinquency task creates 3-day NTV documents.

The downloaded file contains individually named NTV documents to be dragged into resident profiles, and a master file of all NTV documents for printing. The master file is sorted for efficient delivery throughout the property.

Property Name: the name that appears in front of each resident on the delinquency report

Post Date: today's date

Due Date: 3 days after today's date, excluding weekends and holidays

Delinquency Report: run a deliqnuency report

  • Add all charge codes
  • Deselect "primary residents only"
  • Select "notice" and "current" lease statuses
  • Copy and paste the resulting report into the text box
