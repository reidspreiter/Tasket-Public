Property Walk

The property walk task creates a map for efficient property walking.

Enter individual units or select entire buildings at once. If you'd like to pull keys for each unit on the map, select Add Keys. If you'd prefer to pull your keys in property-walk-order instead of alphabetical, select Pull In Order.

If you decide to pull keys, they will be included on the property map, which you can print and hold while you grab keys.
